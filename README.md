# CryptoConquest Overview

CryptoConquest is a decentralized, blockchain-based strategy game inspired by the classic board game Risk. Players own territories, armies, and collectible cards as blockchain tokens (NFTs and fungible tokens) on the Base blockchain, enabling true ownership, secure trading, and provably fair combat with verifiable on-chain randomness.

The gameplay centers on territorial conquest, resource management, and alliance building in a persistent, player-driven world.
